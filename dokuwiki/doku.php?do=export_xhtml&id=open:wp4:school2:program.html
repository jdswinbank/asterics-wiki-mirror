Program

It is important to have installed beforehand in your laptops the VO-tools to be used in the School, namely:

  • ALADIN (Download the Aladin4Asterics.jar file and execute it from a terminal using “java -Xmx2048m -jar Aladin4Asterics.jar”)
  • TOPCAT (Download the topcat-full.jar file and execute it from a terminal using “java -jar topcat-full.jar”)
  • VOSA (Web server. You don't need to install anything)
  • STILTS (Download the stilts.jar file and execute it from a terminal using “java -jar stilts.jar”)
