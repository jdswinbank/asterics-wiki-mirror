Program

It is important to have installed beforehand in your laptops the VO-tools to be used in the School, namely:

  • ALADIN (Download the Aladin.jar file and execute it from a terminal using “java -Xmx2048m -jar Aladin.jar”). Official version: v10
  • TOPCAT (Download the topcat-full.jar file and execute it from a terminal using “java -jar topcat-full.jar”). Official version: v4.6-2
    • See detailed instructions under http://www.star.bris.ac.uk/~mbt/topcat/
  • STILTS (Download the stilts.jar file and execute it from a terminal using “java -jar stilts.jar”). Official version: v3.1-5
    • See detailed instructions under http://www.star.bris.ac.uk/~mbt/stilts/
  • VOSA (Web server. You don't need to install anything). Official version: v6.0
  • Java 1.8 or a later version has to be installed on your system.
    • To check the Java installation from a terminal, type the command line: java -version
    • You can download the latest JRE version from: https://www.java.com/en/download/
