Program

It is important to have installed beforehand in your laptops the VO-tools to be used in the School, namely:

  • ALADIN (Download the AladinBeta.jar file and execute it from a terminal using “java -Xmx2048m -jar AladinBeta.jar”)
  • TOPCAT (Download the topcat-full.jar file and execute it from a terminal using “java -jar topcat-full.jar”)
  • VOSA (Web server. You don't need to install anything)
  • STILTS (Download the stilts.jar file and execute it from a terminal using “java -jar stilts.jar”)
  • VOSpec (Download the VOSpec_6.7.jar file and execute it from a terminal using “java -jar VOSpec_6.7.jar”)
  • SPLAT-VO (Download the splat-vo-20150712.jar file and execute it from a terminal using “java -jar splat-vo-20150712.jar”). Note that, unlike ALADIN, TOPCAT or STILTS, the splat-vo-20150712.jar file does not launch the tool but an installation program. Once the installation has been completed, you should launch the tool with a command like this:
    • $XXXX/splat-vo/bin/splat/splat&
