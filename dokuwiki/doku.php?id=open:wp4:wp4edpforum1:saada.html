Saada

Saada transforms a set of heterogeneous FITS files or VOtables of various categories (images, tables, spectra, …) into a powerful database deployed on the Web, which operates VO services (SSA, SCS, SIS, and TAP) as well. Exposed data can be linked to each other in order to provide a global view on complex data sets.
