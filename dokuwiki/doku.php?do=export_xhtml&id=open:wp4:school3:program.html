Program

It is important to have installed beforehand in your laptops the VO-tools to be used in the School, namely:

  • ALADIN (Download the Aladin.jar file and execute it from a terminal using “java -Xmx2048m -jar Aladin.jar”). Official version: v9.033b
  • ALADINV10 (Download the Aladinv10.jar file and execute it from a terminal using “java -Xmx2048m -jar Aladinv10.jar”). Aladin V10 Avant Premiere
  • TOPCAT (Download the topcat-full.jar file and execute it from a terminal using “java -jar topcat-full.jar”). Official version: v4.5
  • VOSA (Web server. You don't need to install anything). Official version: v5.1
  • STILTS (Download the stilts.jar file and execute it from a terminal using “java -jar stilts.jar”). Official version: v3.2-1
  • SPLAT-VO (Download the splat-vo.jar file and execute it from a terminal using “java -jar splat-vo.jar”). Official version: 3.11. Note that, unlike ALADIN, TOPCAT or STILTS, the splat-vo.jar file does not launch the tool but an installation program. Once the installation has been completed, you should launch the tool with a command like this:
    • $XXXX/splat-vo/bin/splat/splat&
